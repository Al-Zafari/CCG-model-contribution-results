# CCG Model Contribution Results

This repository contains the configuration files and chain outputs for several cosmological model contribution runs.

## Models

| Folder | Model |
|---|---|
| BA | BA |
| CEDE | CEDE |
| cplb | CPL-b |
| cplc | CPL-c |
| exp | Exponential |
| LOG | Logarithmic |
| PEDE | PEDE |

## Dataset combinations

| Run folder | Dataset combination |
|---|---|
| Run1_PP+DESI | PP + DESI |
| Run2_CMB-SPA | CMB-SPA |
| Run3_CMB-SPA+PP+DESI | CMB-SPA + PP + DESI |
| Run4_PPS+DESI | PPS + DESI |
| Run5_CMB-SPA+PPS+DESI | CMB-SPA + PPS + DESI |

Each run folder contains the corresponding `.yaml` configuration file and the output chains in the `chains/` directory.
